# Neural-Networks-in-Computer-Vision


🤖Implementation Description

The project was developed in Python. It is designed to run in Google Colab and is divided into two distinct queries (notebooks):

1. Comparing Basic Architectures (ComparingBasicNNArchitectures)
   Development and comparison of two different architectures on the MNIST dataset:
   
   Deep Neural Network (DNN) & Convolutional Neural Network (CNN):
   Built using Keras, as provided through TensorFlow.
   
   Architecture:
   Both models incorporate ReLu activation functions in all hidden layers, use at least one Dropout layer to prevent overfitting, and end with a Softmax output layer.

   Training & Evaluation:
   K-Fold (or StratifiedKFold) is used to create 6 smaller datasets, and the training utilizes an independent validation set.

   Export:
   The best models (based on test set performance across all 6 experiments) are saved as .h5 files, and performance metrics are recorded in erotima1.csv.

2. Transfer Learning (evaluatingTransferLearningUsability)
   Applying Transfer Learning techniques to adapt the best models from the previous query to the Fashion-MNIST dataset to understand its utility and limitations:

   Models From Scratch:
   Creation and training of a new CNN and DNN (CNN_from_scratch and DNN_from_scratch) on the new dataset for 150 epochs.

   Transfer Models:
   Loading the saved .h5 models from query 1 (named CNN_transfer and DNN_transfer). Their training is implemented in two stages:
      Freezing: Training exclusively the final classification layer for 40 epochs on the train set, keeping the previous layers frozen.
      Fine-Tuning: Unfreezing the layers and training the entire architecture end-to-end for 20 epochs.

   Evaluation & Export: The process takes place within a K-Fold loop (where models are initialized/loaded within the fold loop). Results (Accuracy, Precision, Recall, F1 score) are exported to the erotima2.csv file, resulting in a dataframe with 48 rows plus 1 for the headers.


🛠️Requirements and Installation

All required libraries can be installed via pip. To run the code, follow these steps:

Open the .ipynb files in Google Colab.

Enable hardware acceleration (Hardware Accelerator -> GPU) for optimal training times.

Attention: To run the second notebook (evaluatingTransferLearningUsability), ensure you have uploaded the .h5 files generated by the first query to the Colab environment.

Execute the cells sequentially. The exported .csv files will be generated automatically in your environment.

The assignment focuses on using Deep Learning models for image classification and utilizing Transfer Learning to apply existing models to a new dataset.
